# Java Swing Radio Button Demo

## Description

This project is a Java Swing application that demonstrates the use of radio buttons. Users can select one pet from five available options. When a pet is selected, the corresponding image is displayed and a message dialog shows the selected pet.

## Features

- Five Radio Buttons
- Displays pet images
- Shows a message dialog
- Built using Java Swing
- Runs in Apache NetBeans IDE
